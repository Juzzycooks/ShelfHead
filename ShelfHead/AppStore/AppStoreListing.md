## Description

```
ShelfHead is a streamlined iOS player for your self-hosted Audiobookshelf server. It connects directly to the server you run — no accounts, no middlemen, no ads.

Built for people who just want to listen: a clean, focused interface, fast browsing, and reliable playback that remembers exactly where you left off.

FEATURES
• Connect to your own Audiobookshelf server
• Browse books, authors, series, collections, and playlists
• Variable playback speed, with per-book and global defaults
• Chapter navigation and a sleep timer
• Download titles for offline listening
• Progress, bookmarks, and finished status sync back to your server
• Background playback and full lock-screen controls
• CarPlay support for listening on the road
• A clean, native design that feels at home on iOS

ShelfHead requires an existing Audiobookshelf server (audiobookshelf.org). The app does
not include or provide any audiobook content — it plays the library you already host.

Your data stays yours: ShelfHead has no analytics, no tracking, and no advertising, and it
talks only to the server you configure.
```

## App privacy ("Data Collection") answers in App Store Connect

Select **"Data Not Collected."** ShelfHead stores credentials and settings only on the
device (Keychain / app preferences) and sends data only to the user's own server; the
developer receives nothing. This matches the included `PrivacyInfo.xcprivacy`.
